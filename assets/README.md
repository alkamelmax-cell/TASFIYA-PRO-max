# Assets Directory

هذا المجلد يحتوي على الملفات المساعدة للتطبيق:

## الملفات المتوفرة:

### الأيقونات والشعارات:

1. **icon.ico** - أيقونة التطبيق للويندوز (متعددة الأحجام)
2. **icon.png** - أيقونة التطبيق بصيغة PNG (256x256)
3. **logo-tasfia-pro.svg** - شعار TASFIA PRO بصيغة SVG (قابل للتحرير)

### الشعار الجديد (TASFIA PRO):

تم اعتماد شعار TASFIA PRO الجديد كشعار رسمي للتطبيق، والذي يتضمن:
- إطار دائري أزرق
- ميزان العدالة مع رموز المال والرسوم البيانية
- نص TASFIA PRO

## ملفات الشعار في مجلد src/web-dashboard/assets/:

- **logo-tasfia-pro.png** - الشعار الرئيسي (512x512)
- **icon-512.png** - أيقونة 512x512
- **icon-192.png** - أيقونة 192x192
- **favicon.png** - أيقونة المتصفح (64x64)

## طريقة تحديث الشعار:

1. تعديل ملف `assets/logo-tasfia-pro.svg` باستخدام أي محرر SVG
2. تشغيل سكريبت التوليد:
   ```bash
   node generate-logo.js
   node generate-ico.js
   ```
