# Production Delivery Checklist - Tasfiya Pro v5.0.1

هذه القائمة مخصصة لتسليم نسخة `Production` من تطبيق `تصفية برو` قبل النشر أو التسليم لأي عميل.

## 1. فحص البناء

- [ ] تأكد من نجاح `npm run lint`
- [ ] تأكد من نجاح `npm test`
- [ ] تأكد من نجاح `npm audit --omit=dev`
- [ ] تأكد من نجاح `npm run build-clean`
- [ ] تأكد من وجود الملفات التالية داخل `dist/`
- [ ] `تصفية برو - Tasfiya Pro-5.0.1-x64.exe`
- [ ] `تصفية برو - Tasfiya Pro-5.0.1-portable.exe`
- [ ] `latest.yml`
- [ ] `win-unpacked/`

## 2. فحص التثبيت

- [ ] جرّب تثبيت نسخة `x64.exe` على جهاز Windows نظيف
- [ ] تأكد من ظهور اسم التطبيق الصحيح في المثبّت وقائمة ابدأ
- [ ] تأكد من إنشاء اختصار سطح المكتب
- [ ] تأكد من إمكانية تشغيل التطبيق بعد التثبيت مباشرة
- [ ] تأكد من إزالة التطبيق بدون حذف بيانات العميل إلا إذا كان ذلك مطلوبًا
- [ ] جرّب تشغيل النسخة `portable.exe` بدون تثبيت

## 3. فحص الدخول والصلاحيات

- [ ] تأكد من إمكانية تسجيل الدخول بحساب مدير
- [ ] تأكد من عدم وجود حسابات افتراضية غير مصرح بها في بيئة العميل
- [ ] تأكد من تغيير كلمات المرور الأولية قبل التسليم
- [ ] جرّب صلاحيات مستخدم محدود
- [ ] تأكد من إخفاء التبويبات والعمليات غير المصرح بها
- [ ] تأكد من عمل القفل التلقائي وانتهاء الجلسة حسب الإعدادات

## 4. فحص البيانات الأساسية

- [ ] إضافة فرع جديد
- [ ] إضافة كاشير جديد
- [ ] إضافة محاسب جديد
- [ ] إضافة جهاز صراف وربطه بالفرع
- [ ] إضافة صندوق إن كان العميل يستخدم الصناديق
- [ ] التأكد من ظهور البيانات في القوائم والنماذج والتقارير

## 5. فحص دورة التصفية

- [ ] إنشاء تصفية جديدة كاملة
- [ ] إضافة عمليات بنكية
- [ ] إضافة عمليات نقدية
- [ ] إضافة مبيعات آجلة
- [ ] إضافة مقبوضات عملاء
- [ ] إضافة مرتجعات
- [ ] إضافة موردين إذا كانت مستخدمة
- [ ] حفظ التصفية
- [ ] استدعاء التصفية لاحقًا والتأكد من تطابق البيانات
- [ ] تعديل التصفية ثم حفظها مرة أخرى

## 6. فحص الرسوم البنكية

- [ ] فتح `الإعدادات > إعدادات التقارير`
- [ ] إضافة قاعدة رسوم بنك واحدة على الأقل
- [ ] تجربة نسبة دقيقة مثل `0.008`
- [ ] التأكد من بقاء النسبة كما أُدخلت بدون تقريب غير مرغوب
- [ ] تجربة نوع عملية عربي مثل `مدى`
- [ ] تجربة نوع عملية مختلط مثل `مدى(mada)`
- [ ] تجربة نوع عملية مختلط مثل `Visa (فيزا)`
- [ ] فتح التقرير التحليلي واختيار `مع الرسوم`
- [ ] التأكد من ظهور الأعمدة:
- [ ] `المبلغ قبل الرسوم`
- [ ] `الرسوم البنكية`
- [ ] `ضريبة الرسوم`
- [ ] `المبلغ بعد الرسوم`
- [ ] تجربة خيار `بدون رسوم`
- [ ] تجربة خيار `تذكر آخر اختيار`

## 7. فحص التقارير

- [ ] تقرير التصفيات العام
- [ ] التقرير التحليلي لأجهزة الصراف
- [ ] تقارير الأداء
- [ ] تقارير المبيعات الآجلة
- [ ] دفتر العملاء
- [ ] دفتر الموردين
- [ ] التحقق من صحة المجاميع والمرشحات
- [ ] التحقق من الترتيب والبحث والصفحات

## 8. فحص التصدير والطباعة

- [ ] تصدير Excel للتقارير الأساسية
- [ ] تصدير Excel للتقرير التحليلي
- [ ] معاينة A4
- [ ] طباعة A4
- [ ] طباعة حرارية إن كانت ضمن نطاق العميل
- [ ] التأكد من اسم الشركة في الترويسة
- [ ] التأكد من الهوامش والخطوط والطابعة الافتراضية

## 9. فحص النسخ الاحتياطي والاستعادة

- [ ] إنشاء نسخة احتياطية
- [ ] حفظ النسخة في مسار خارجي
- [ ] استعادة النسخة على بيئة اختبار
- [ ] التأكد من رجوع الفروع والكاشيرين والأجهزة والتصفيات
- [ ] التأكد من سلامة `system_settings`

## 10. فحص المزامنة والويب

- [ ] التحقق من عنوان الخادم الفعلي المستخدم عند العميل
- [ ] اختبار سحب الطلبات من الويب
- [ ] اختبار دفع البيانات المحلية
- [ ] اختبار التعامل مع انقطاع الشبكة
- [ ] التأكد من عدم تكرار البيانات بعد إعادة المزامنة
- [ ] التأكد من تطابق أنواع العمليات القادمة من الويب مع قواعد الرسوم

## 11. فحص الأداء والاستقرار

- [ ] تشغيل التطبيق أكثر من مرة بدون أخطاء بدء
- [ ] فتح وإغلاق النوافذ الرئيسية
- [ ] تجربة بيانات كثيرة نسبيًا
- [ ] التأكد من عدم تجمد الواجهة أثناء العمليات المتكررة
- [ ] مراجعة ملفات السجل إن كانت معتمدة لدى العميل

## 12. فحص الاعتماد التجاري

- [ ] اسم المنتج صحيح في التثبيت
- [ ] الإصدار صحيح `5.0.1`
- [ ] الأيقونة صحيحة
- [ ] بيانات المطور صحيحة
- [ ] لا توجد رسائل تجريبية أو نصوص قديمة داخل المثبّت
- [ ] لا توجد بيانات عميل تجريبية داخل النسخة النهائية

## 13. متطلبات التسليم للعميل

- [ ] ملف التثبيت
- [ ] الملف المحمول إذا لزم
- [ ] دليل تشغيل مختصر
- [ ] بيانات الدخول الأولية
- [ ] سياسة النسخ الاحتياطي
- [ ] آلية الدعم الفني
- [ ] شخص مسؤول من جهة العميل للتجربة والاستلام

## 14. اعتماد نهائي

- [ ] تمت التجربة على جهاز نظيف
- [ ] تمت مراجعة الصلاحيات
- [ ] تمت مراجعة التقارير والطباعة
- [ ] تمت مراجعة النسخ الاحتياطي والاستعادة
- [ ] تمت مراجعة الرسوم البنكية
- [ ] تمت مراجعة المزامنة
- [ ] النسخة جاهزة للتسليم
