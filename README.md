# 🧾 تصفية برو - Tasfiya Pro v5.0.1

نظام تسوية نقدية احترافي مع دعم كامل للغة العربية وواجهات RTL.

## ✨ المميزات الرئيسية

- ✅ إدارة التسويات بين المبيعات والإيصالات
- ✅ تتبع معاملات البنك والنقد والمبيعات بالآجل
- ✅ واجهة عربية كاملة (RTL)
- ✅ طباعة وتصدير PDF
- ✅ نسخ احتياطي تلقائي للبيانات
- ✅ قاعدة بيانات SQLite آمنة
- ✅ دعم البحث والتصفية المتقدم

## 🚀 الاستخدام السريع

### المتطلبات
- Windows 10/11
- Node.js 16+
- npm 8+

### تثبيت وبناء التطبيق

```bash
# تثبيت المتطلبات
npm install

# بناء التطبيق نظيف
npm run build

# أو استخدم سكريبت البناء الشامل
npm run build-clean
```

### تشغيل التطبيق

```bash
# تشغيل الإصدار المثبت
npm start

# أو في وضع التطوير
npm run dev
```

## 👤 بيانات الدخول الافتراضية

بيانات الدخول الافتراضية متاحة في **بيئة التطوير فقط**.

| الحقل | القيمة |
|------|--------|
| اسم المستخدم | admin |
| كلمة المرور | admin123 |

في بيئة الإنتاج، لن يتم إنشاء بيانات `admin/admin123` تلقائياً.
لتهيئة أول حساب مسؤول في الإنتاج، شغّل التطبيق مع المتغيرات:

```bash
INITIAL_ADMIN_PASSWORD=your_secure_password
# اختياري:
INITIAL_ADMIN_USERNAME=admin
INITIAL_ADMIN_NAME="مدير النظام"
```

⚠️ **تنبيه أمني**: غيّر كلمة المرور مباشرة بعد أول تسجيل دخول.
